# 工作空间说明

欢迎来到这个工作空间！这是一个用于存放C++项目的文件夹。

## 目录结构

这个工作空间的目录结构如下：

- `src/`：存放所有的源代码文件
- `include/`：存放所有的头文件
- `bin/`：存放编译生成的可执行文件
- `docs/`：存放项目文档和说明文件

## 如何编译和运行

1. 在终端中进入项目根目录：`cd /home/boran/Program/Cpp`
2. 创建一个名为`build`的文件夹：`mkdir build`
3. 进入`build`文件夹：`cd build`
4. 使用CMake生成Makefile：`cmake ..`
5. 编译项目：`make`
6. 运行可执行文件：`./bin/your_executable`
